RUnning di local tdk disarankan krn saya melihat banyak sekali issue dg python configuration setiap komputer yang berbeda2. Tapi tetap saya catat sbb:  

- Install python *saya menggunakan v3.14.5 MSI installer
- Lalu buka terminal dan ketik `python` lalu enter

![Python]()  

- Setelah itu install pip untuk install `ipykernel` karena vscode butuh itu
![Ipykernel]()  

- Install VSCode
- Install extension vscode untuk jupyter notebook dan juga python
![PythonExt]()  
![JupyterExt]()  

Info troubleshoot:
Jika terjadi stuck atau code tidak running coba untuk `restart kernel` atau `restart vscodenya`

### Run di google collab  
Download repo ini sebagai zip, ekstrak, lalu buka google collab dan pilih `upload`, pilih file `ipynb` yang ingin di run, lalu klik `run all`
